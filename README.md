# Insurance Benefit Request Management System

A Domain-Driven Design (DDD) .NET Core project for managing insurance benefit requests. 
The system uses EF Core for data access, PostgreSQL as the database, and follows best practices for aggregate roots and entity relationships.


## Technologies

- .NET Core 8
- C#
- Entity Framework Core
- PostgreSQL
- Domain-Driven Design (DDD)
- Repository Pattern
- Aggregate Root & Value Objects
