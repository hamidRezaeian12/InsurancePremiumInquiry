# InsurancePremiumInquiry
A Domain-Driven Design (DDD) .NET Core project for managing insurance benefit requests, using EF Core and PostgreSQL.

## Technologies

- .NET Core 8
- C#
- Entity Framework Core
- PostgreSQL
- Domain-Driven Design (DDD)
- Repository Pattern
- Aggregate Root & Value Objects
- API Versioning
- Mediator
- Clean Architecture
